# pyenv-wrapper

Use and configure pyenv as a virtualenvwrapper

## Install

With [fisherman]

```
fisher robsonpeixoto/pyenv-wrapper
```

## Notes

`pyenv_wrapper` was built for the latest fish.

## Usage

### List all virtualenvs

```fish
lsvirtualenv
```

### Create a virtualenv

```
mkvirtualenv 2.7.11 myvenv
```

### Remove a virtualenv

```
rmvirtualenv myvenv
```

### Activate a virtualenv

```
workon myvenv
```
